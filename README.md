# tinywebplayground

A home for small ideas that don't fit anywhere else — a personal playground of tiny, self-contained web toys, tools, and experiments. Built in the spirit of the small web.

## A few of the projects inside

- **HowLongToWatch** — Search movies and TV shows, build a watchlist, see your total watch time.
- **History of Rock** — An interactive diagram of rock music's evolution, inspired by School of Rock.
- **Tiny Eurorack** — A simplistic modular synth with oscilloscope visualization.
- **HAIKU** — The world's most advanced AI haiku generator. Probably.
- **Tabletop Dice Roller** — Keyboard-first dice roller - also available for Raycast.
- **Human Maintenance Guide** — A reference for how often to do the things that keep life running.

…and more on the hub page.

## Stack

Astro 5 + React 19 islands, Zustand for state, plain CSS with CSS variables, deployed on Netlify.

## Run it locally

```sh
npm install
npm run dev
```
